# 🚀 IBM Data Analyst Capstone Project – Technology Trends Analysis

## 📌 Project Overview

This project was completed as the final capstone of the **IBM Data Analyst Professional Certificate**. It demonstrates an end-to-end data analytics workflow using real-world datasets to analyze **current technology usage, future technology trends, job market demand, and developer demographics**.

The project includes data collection, data cleaning, exploratory data analysis (EDA), interactive dashboard creation, and business insight generation to support data-driven decision-making.

---

## 🎯 Objectives

- Analyze current technology usage among developers.
- Identify future technology trends and preferences.
- Explore job market demand for technical skills.
- Understand developer demographics.
- Build interactive dashboards for better decision-making.

## 🔍 Key Insights

- Python, JavaScript, SQL, and PostgreSQL continue to dominate the technology landscape.
- AWS and Microsoft Azure remain the leading cloud platforms.
- Developers show increasing interest in modern technologies, cloud computing, and scalable databases.
- The majority of survey respondents belong to the **25–34 years** age group, representing a young and highly skilled developer workforce.
